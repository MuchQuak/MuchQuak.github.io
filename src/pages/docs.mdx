<h1 class="text-2xl pb-3">Ways to Create Lists</h1>

There are various ways to create a list. <br /> <br />

<dl>
  <dt>  <b> 1. From the Search Results Page </b> </dt> <br />
  <dd>
    Upon viewing the results of a search, a user can click on the plus button on
    the upper righthand corner of a tree card. From here, a modal called "My
    Lists" should appear and the user should click on "Add to New List".
  </dd>

  <dt> <b> 2. From the "My Lists" Page </b> </dt> <br />
  <dd>
    The "My Lists" page can be navigated to by clicking on the user profile
    dropdown button in the upper right corner of the website. Then select "My
    Lists". Upon viewing this page, click on the "Create a list" green button on
    the left hand side of the screen.
